# РНП — Ресурсы, нужды, помощь

## Нужна помощь с

- Выход на школы-партнёры в регионах (Московская область, СПб, Екатеринбург). Интересуют директора школ и завучи, открытые к пилотам.
- Интеграция с LMS МосОбр.ру — нужна теплая рекомендация к техкоманде платформы.
- Упаковка кейса для фонда: поднять следующий раунд инвестиций в Q3 2025.
- Договорённости с топ-5 блогерами в Telegram, работающими на аудитории подростков (не на родителей) — хотим протестировать D2C-канал в 10-11 классах.
- Консультация по юридической упаковке B2B-договоров с государственными школами (94-ФЗ, 44-ФЗ).

## Готовы делиться

- Опыт запуска ВК-таргета в EdTech (связки, креативы, сегментация)
- Методички для подростков по ОГЭ/ЕГЭ (готовы отдать портфельным компаниям бесплатно)
- Платформенная часть тренажёров — можно обсуждать white-label для B2B-клиентов других портфельных
- Доступ к тьюторской сети (при совпадении профилей)

## Ресурсы

- Команда разработки на Go / Python
- Собственный CRM для работы с родителями (интеграция с amoCRM + собственная аналитика)
- 40 тыс активных родителей в базе (mail/tg) для кросс-промо
- Сеть из 14 тьюторов (математика, русский, физика, химия, биология, обществознание, история)
