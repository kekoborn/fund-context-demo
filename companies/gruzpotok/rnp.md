# РНП — Ресурсы, нужды, помощь

## Нужна помощь с

- Контакты в крупных FMCG-компаниях (100-300 машин) — для прямого захода
- Региональные партнёры в Новосибирске (GPS-трекеры, 1С-интеграторы)
- Юридическая помощь по регуляторике электронных путевых листов (ЭТрН)
- Маркетплейс-грузоперевозки: нужны контакты транспортных ассоциаций
- PR в транспортной прессе ("Rail-News", "Autostat", "Грузовики Weekly")

## Готовы делиться

- Опыт прямых B2B-продаж с холодным outbound
- Методология интеграции GPS-трекеров от разных поставщиков (Navixy, Scout, Navi)
- Кейсы клиентов — по 3 отраслям (FMCG, строительство, ритейл)
- Партнёрский канал (сеть поставщиков GPS) — открыт для кросс-рекомендаций

## Ресурсы

- Платформа (веб + мобильное приложение для водителей)
- База 186 активных клиентов + 240 в тёплой воронке
- Команда разработки с экспертизой по ГЛОНАСС/GPS
- Данные по рынку: бенчмарки по логистике в FMCG, строительстве, ритейле
