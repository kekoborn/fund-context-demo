# Тактический план Q2 2025

## 1. Utilization 19% → 26%

**Цель:** основной драйвер retention.

- Запуск HR-программы "Запустите сервис за 30 дней" — чек-лист для HRD
- Регулярные email-рассылки сотрудникам (темы месяца, истории)
- Интеграция с корпоративными мессенджерами (Teams, Slack) — бот-напоминания
- Запуск челленджей внутри компаний ("30 дней без стресса")

## 2. Запуск продукта "Культура заботы"

**Цель:** первые 8 компаний на курсе к концу июня.

- Методичка для HR — 6-недельный курс внедрения культуры заботы
- Маркетинговая упаковка и запуск 15 мая
- Партнёрство с 2 HR-tv спикерами для промо
- Цена — доп 25 тыс ₽/мес к базовому тарифу

## 3. Enterprise-направление

**Цель:** 2 enterprise-пилота к концу июня.

- Найм 2 enterprise AE до 15 мая
- Продукт для enterprise — выделенный SLA, персональный CSM, кастомный HR-дашборд
- Целевой список из 30 компаний — персональные email-кампании
- Первые 5 встреч с CHRO до конца апреля (есть тёплые контакты)

## 4. Партнёрство со страховой

**Цель:** 1 подписанный партнёр к концу Q2.

- Переговоры с тремя страховыми (СОГАЗ, Ренессанс, Ингосстрах)
- Пилот-продукт: сервис Менторы как часть ДМС-пакета
- Юридическая проработка схемы
