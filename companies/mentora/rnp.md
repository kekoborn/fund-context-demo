# РНП — Ресурсы, нужды, помощь

## Нужна помощь с

- Контакты CHRO в компаниях 1000+ (IT, консалтинг, производство)
- Тёплые переговоры с ДМС-департаментами страховых ("СОГАЗ", "Ренессанс", "Ингосстрах")
- Стратегическая консультация по enterprise-каналу (как выстроить процесс под 3-6-месячный цикл)
- PR-партнёры для публикаций в HR-медиа (HR-tv, HR-Journal, HR-pulse)
- Юридическая помощь по корпоративным данным и персональной информации сотрудников

## Готовы делиться

- Методика HR-онбординга сервиса (6-недельный процесс внедрения)
- Данные по utilization по сегментам (обезличенные)
- Сеть психологов (возможен white-label или партнёрский доступ)
- Контент-библиотека — 12 курсов по ментальному здоровью (в обмен на кросс-промо)

## Ресурсы

- 312 проверенных психологов в сети
- Платформа с мобильным приложением (iOS/Android)
- HR-дашборд с аналитикой
- 78 корпоративных клиентов и 31,5 тыс покрытых сотрудников как база для кросс-продуктов
