# fund-context-demo

Демонстрационный markdown-контекст для fund-dashboard `/demo`.

Структура полностью повторяет реальный `fund-context/`, но содержит **смоделированные данные** по 5 вымышленным портфельным компаниям за период январь-апрель 2025.

## Компании

| slug         | Имя            | Профиль                        |
|--------------|----------------|--------------------------------|
| nova-teach   | NovaTeach      | EdTech, онлайн-школа для 5-11 классов |
| zeleny-obed  | Зелёный Обед   | FoodTech, доставка ЗОЖ-питания |
| finpulse     | FinPulse       | B2B FinTech для МСП            |
| mentora      | Ментора        | Корпоративное ментальное здоровье |
| gruzpotok    | ГрузПоток      | SaaS для среднего бизнеса (логистика) |

## Для чего это

Чтобы внешним людям можно было показать `/demo` без риска утечки реальных данных клиентов.
